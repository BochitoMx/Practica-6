FRONTEND


Es la parte del desarrolo web que se encarga del diseño y la interacción visual que los usuarios ven y experimenta en el navegador así como colores, fondos, tamaños, animaciones y efectos.


FUNDAMENTOS:


  • HTML: Lenguaje de marcado que estructura el contenido de una página web mediante etiquetas y también como encabezados, enlaces, imágenes, párrafos y listas, entre otros más elementos.

  • CSS: Define el diseño visual de una página web (colores, fuentes, márgenes, diseños bruscos, tipografías, layout, etc.)

  • JavaScript: Facilita la creación de interactividad y dinamismo de las páginas web, permitiendo formularios, manipular, modificar elementos HTML, estilos en tiempo real, también manejando eventos del usuario como clics, movimiento del ratón o teclas presionadas.
